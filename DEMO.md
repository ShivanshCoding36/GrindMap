### How to Access the Demo

1. **Start the application:**
   ```bash
   cd frontend
   npm start
   ```

2. **Click "View Demo" button** on the main page

3. **Explore the features:**
   - View sample statistics from LeetCode, CodeForces, and CodeChef
   - Click on platform cards to expand and see detailed information
   - Explore the submission heatmap
   - Check today's activity tracker

### Demo Features

The interactive demo showcases:

- **Multi-platform tracking** with realistic sample data
- **Circular progress indicators** for each platform
- **Expandable cards** with detailed statistics
- **Activity heatmap** showing submission patterns
- **Daily activity tracker** for consistency monitoring
