# Screenshots Directory

This directory contains visual assets for the GrindMap project.

## Contents

Add screenshots here to showcase:

- Dashboard overview
- Platform statistics
- Activity heatmaps
- Progress tracking
- Demo mode

## Guidelines

- Use PNG format for images
- Keep file sizes under 500KB
- Use descriptive filenames
- Maintain consistent dimensions

See [SCREENSHOT_GUIDE.md](../../SCREENSHOT_GUIDE.md) for detailed instructions.
